A lightweight, browser-based tool for sentence-by-sentence shadowing practice.  
No installation, no server, no internet required after download.

## Features

- Paste any English script → auto-split into sentences
- Click any sentence to play it instantly
- Auto-play mode for continuous shadowing
- Adjustable speed (0.5× – 1.5×) and pause between sentences
- Keyboard shortcuts: `Space` play/pause, `←` `→` prev/next sentence
- Uses browser built-in Text-to-Speech (Web Speech API)

## How to Use

1. Download `shadow-reader.html`
2. Open it in any modern browser (Chrome, Edge, Safari, Firefox)
3. Paste your script and click **载入文稿**
4. Click any sentence to hear it, or enable **自动顺读** for auto-play

## Browser Support

Works best in Chrome or Edge. Firefox has limited TTS voice support.
